What is HTML?

  • Short hand for Hyper Text Markup Language
  • Basic building block/skeleton of a website

Web Browsers

Here, web browsers (Chrome, Edge, Mozilla, Safari) are going to play an important role. As web browsers is the one, going to read the HTML documents and display them correctly. It is not gong to display the HTML tags, but will use them to determine how to display an HTML document.

Basic HTML template

<!DOCTYPE html>
<html>
<head>
<title>Title of Page </title>
</head>
<body>
<h1>First Heading</h1>
<p>First Paragraph</p>
</body>
</html>

Come and let's explore this HTML template together


  • <!DOCTYPE html> shows that the document is HTML5
  • <html> - </html> indicates that the webpage is written in HTML5. This is the begining and end of a HTML document. Whole markup comes in between these begining and ending tags.
  • <html lang="en"> "lang" attribute defines the language of the document. "en" stands for english. So here 'html lang ="en"' defines that the language of this HTML document is english.
  • <head> element is a container for metadata (data about data) and contains all the meta information about the HTML page.
  • <title> tag defines the title of the HTML page (which is shown in the browser's title bar or in the web page's tab).
  • <body> container for all the visible page contents, lile headings, links, tables etc.
  • <h1> tag defines the heading of a webpage.
  • <p> tag defines the paragraph of a webpage.

How to get the Source Code of a Website?


HTML Source Code

  • Step-1 Right-click in an HTML page.
  • Step-2 Select "View Page Source" (in Chrome) or "View Source" (in Edge), or similar in other browsers.

    • This will open a window containing the HTML source code of the page.

HTML Element Inspection

  • Step-1 Right-click on an element (or a blank area)
  • Step-2 Click on "Inspect" or "Inspect Element"

    • And here both the HTML and the CSS part of a webpage and you can edit the HTML & CSS on-the-fly in the Elements or Styles panel that opens.

HTML Elements

An HTML element is defined by a start tag, content, and an end tag. It is the basic building block of HTML page. HTML elements are delineated by tags, which is written using angle brackets.

<tag> Content </tag>

Example

<h1>First Heading</h1>

<h1>First Paragraph</h1>
